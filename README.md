Star Wars REST API with Spring Boot
========================================

Java 8 and MongoDB is needed to run this project.

Clone
--------

```sh
git clone https://github.com/AlexJCosta/starwars-be.git
```

## Instructions

To compile (also runs unit tests)

```
mvn package
```

## To run manually

```
mvn spring-boot:run
```

## Access

```
http://localhost:3000/
```
